About this project

The purpose of this app is to make it easier for people to compare house price statistics of Barcelona neighbourhoods. In total, Barcelona has 10 districts (distritos) and 73 neighbourhoods (barrios). The level of analysis in this app is always the neighbourhood.

The data can be visualised in two ways. In the "Color maps" tab, you can see a choropleth map based on the variable you choose (e.g., average selling price, average monthly rent). A choropleth map uses different shades of the same color to represent an aggregate summary of a variable. For example, a larger average selling price has a darker shade than a lower average selling price.

In the "Compare neighbourhoods" tab, you can filter and manually pick which neighbourhoods you are interested in. You can then generate a table that gives you some summary statistics about the selected neighbourhoods.

Data

The data for this project were manually collated from publicly available sources. I used this site from the Barcelona City Council to find information (e.g., sales price, monthly rent) about the neighbourhoods in Barcelona. I then combined various pieces of information to create new variables and added the neighbourhoods' shapefiles. The result of this somewhat messy process is a single file (available in the source repository), which I use as the only data input for this Shiny app.

My goal with this project was to build a basic Shiny app and to learn something interesting about topics I like - Barcelona and house prices. Although I try to use accurate numbers, it's possible (likely even) that some numbers you will see in this app are inaccurate, outdated, or suboptimally chosen. Just to give you an example: I collected sales and rent prices of used flats from 2019 — for no other reason that this was simpler than collecting and cleaning a richer data set. In other words, I focused on the technical aspects of the app, and not on the data. This means that those who primarily care about the data should not rely on this app but consult official sources instead.

Motivation

I had two goals with this app. First, to build my first Shiny app and experiment with some technical things (e.g., adding custom CSS, creating a module, playing with reactivity) I haven't tried before. Second, to learn something about Barcelona neighbourhoods and the local housing market.
